DougsPCBDesigns
===============

Doug's mergeKiMods
------------------

- Program merges two KiCad modules (PCB footprint) into a single file.
- Program compares the index tables of the two files and looks for duplicates.

See [Doug's PCB Designs Wiki][myWiki].
